## 📌 Project Overview

**FirebaseChat Android App** is a real‑time mobile messaging application developed using Android (Java) and Firebase. It demonstrates how to implement **real‑time database updates**, **user authentication**, **chat messaging**, and **media handling**.

This project serves as a strong example of **mobile app development with cloud integration**, suitable for portfolios, academic demonstrations, and practical learning.

---

## 🧠 System Architecture

The application follows a **layered architecture**:

### 🔷 Layers

- **UI Layer (Activities & Fragments)**  
  Handles rendering of screens and user interaction.

- **Business Logic Layer**  
  Controls message handling, database interaction, and validation.

- **Data Layer (Firebase)**  
  Manages authentication, real‑time database, and storage.
